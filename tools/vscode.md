## Sorting imports

folder .vscode
file: settings.json

Example for react with [[typescript]]
```json
{

	"[typescript]": {

		"editor.codeActionsOnSave": {

			"source.organizeImports": true

		}

	},

	"[typescriptreact]": {

		"editor.codeActionsOnSave": {

			"source.organizeImports": true

		}

	}

}
```
